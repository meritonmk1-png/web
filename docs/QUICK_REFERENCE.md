# Quick Reference Guide

## How to Switch Languages
1. Look for the "DE | EN" button in the top-right of the navbar
2. Click it to toggle between German and English
3. Your choice is automatically saved

## How to Test Locally
```bash
npm run dev
```
Then visit: http://localhost:5173

## How to Build
```bash
npm run build
```

## How to Deploy to Netlify
```bash
npx netlify deploy --prod --dir=build
```

## Translation Files Location
- English: `src/lib/i18n/en.json`
- German: `src/lib/i18n/de.json`
